Kullanım Şartları

Anka Holding olarak, web sitemizi (www.ankaholding.org) kullanarak, aşağıdaki kullanım şartlarına ve koşullarına uymayı kabul etmektesiniz. Web sitemizi kullanmadan önce bu şartları dikkatlice okumanız önemlidir. Bu kullanım şartları, web sitemizin sağladığı hizmetlerden yararlanırken taraflar arasında bağlayıcı bir anlaşma oluşturur.


1. Genel Bilgiler

Web sitemiz üzerinden sağlanan hizmetler, Anka Holding’e ait olup, sadece kişisel kullanım için sunulmaktadır. Web sitemizi kullanarak, bu kullanım şartlarını kabul etmiş olursunuz. Web sitemizde yer alan tüm içerikler, metinler, grafikler, logolar, görseller ve diğer materyaller Anka Holding’in mülkiyetindedir ve izinsiz kullanılamaz.


2. Kullanıcı Sorumlulukları

Web sitemizi kullanırken şu şartları kabul ediyorsunuz: Web sitesindeki içerikleri sadece kişisel amaçlar için kullanacaksınız. Yasadışı, tehdit edici, zararlı, küfürlü veya müstehcen içerik paylaşmayacaksınız. Web sitesinin işleyişini bozacak veya web sitesine zarar verecek hiçbir işlemde bulunmayacaksınız. Başkalarının haklarını ihlal etmeyecek şekilde web sitesini kullanacaksınız. Web sitesi üzerinden sağlanan herhangi bir hizmeti, tescilli hakları ihlal etmeden ve yasa dışı amaçlarla kullanmayacaksınız.


3. İçerik ve Fikri Mülkiyet Hakları

Web sitemizde bulunan tüm içerikler, Anka Holding’in fikri mülkiyetine aittir. Bu içerikler, yazılım, grafikler, görseller, metinler ve diğer materyaller, Anka Holding’in izni olmadan kopyalanamaz, çoğaltılamaz veya dağıtılamaz. İçeriği kullanmak için önceden yazılı izin almak gerekmektedir.


4. Çerezler (Cookies)

Web sitemiz, kullanıcı deneyimini geliştirmek amacıyla çerezler kullanmaktadır. Çerezler, web sitesi ziyaretinizi analiz etmek ve özelleştirilmiş içerik sağlamak için kullanılır. Çerezleri devre dışı bırakmak için tarayıcı ayarlarınızı değiştirebilirsiniz.


5. Sorumluluk Reddi

Web sitemiz "olduğu gibi" sunulmaktadır ve Anka Holding, web sitesinin kesintisiz veya hatasız olacağına dair herhangi bir garanti vermez. Web sitesi üzerinden sağlanan bilgiler, yanlışlıklar veya eksiklikler içeriyor olabilir. Web sitesini kullanarak, tüm riskin sizin sorumluluğunuzda olduğunu kabul etmektesiniz.


6. Değişiklikler

Anka Holding, bu kullanım şartlarını zaman zaman güncelleyebilir. Değişiklikler web sitemizde yayınlandığı anda geçerli olacaktır. Kullanıcılar, web sitesini her ziyaret ettiklerinde güncellenmiş kullanım şartlarını kabul etmiş sayılırlar.


İletişim

Kullanım şartlarımız ile ilgili herhangi bir sorunuz veya talebiniz varsa, bizimle iletişime geçebilirsiniz. Anka Holding, kullanıcıların gizliliğine saygı duyar ve bu kullanım şartlarına uygun olarak verilerinizi korumak için her türlü önlemi alır.
